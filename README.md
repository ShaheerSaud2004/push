# Khafī (خفي) - The Hidden Word Game

A pass-and-play party game where players try to identify the imposter while keeping a secret word hidden.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open in Expo Go on your device or emulator.

## Features

- Three beautiful UI themes
- Multiple game modes (Standard, Blind Imposter, Double Agent)
- Islamic-themed categories
- Custom category creation
- Smooth animations and haptics
- Local persistence

## Sharing the App

See **[SHARING.md](./SHARING.md)** for detailed instructions on how to share the app with friends!

**Quick share:**
1. Run `npm start`
2. Press `s` for tunnel mode
3. Share the QR code/link with friends
4. They scan with Expo Go app
