Proof of Consent - SMS Opt-In Process

Service: Khafī (Muslim Imposter) - SMS Game Card Delivery

Date: January 2025

Overview

This document demonstrates how users provide explicit consent to receive SMS messages from Khafī (Muslim Imposter) for game card delivery. Users must actively opt-in through the mobile app by selecting SMS delivery and providing their phone number.

Opt-In Flow

1 User initiates game setup

User opens the app and begins setting up a new game (selecting players, game mode, categories).

2 User selects delivery method

User is presented with two options:

  • "Pass and Play" - Traditional method (default)
  • "Send via SMS" - SMS delivery option

Consent Language Displayed:
"By providing your phone number, you consent to receive game cards via SMS. Message and data rates may apply. Reply STOP to opt-out."

3 User enters phone numbers

If user selects "Send via SMS", phone number input fields appear for each player. User must manually enter each phone number.

Key Point: No phone numbers are pre-filled or automatically collected.

4 User confirms and sends

User taps "Send Cards" button to confirm and send game cards via SMS. This action is explicit and user-initiated.

5 SMS sent only after explicit action

Messages are sent only after user completes all steps and taps "Send Cards". No messages are sent automatically or without user action.

Consent Requirements Met

  • ✅ Explicit Opt-In: Users must actively select "Send via SMS" option
  • ✅ Clear Disclosure: Consent language is displayed before phone number entry
  • ✅ User Action Required: Users must manually enter phone numbers
  • ✅ No Pre-Checked Boxes: SMS delivery is opt-in, not opt-out
  • ✅ Opt-Out Instructions: Users are informed they can reply "STOP" to opt-out
  • ✅ Single Purpose: SMS is only used for requested game card delivery
  • ✅ No Marketing: Only transactional game-related messages are sent

Consent Language in App

Text displayed to users when selecting SMS delivery:

"By providing your phone number, you consent to receive game cards via SMS. Message and data rates may apply. Reply STOP to opt-out."

Opt-Out Mechanism

  • Users can reply "STOP" to any SMS message to opt-out
  • Users can reply "HELP" for support information
  • Opt-out is immediate and permanent for that phone number
  • No further messages will be sent after opt-out

Message Content

All SMS messages sent are transactional game cards containing:

  • Player name
  • Assigned word
  • Category
  • Role (Imposter/Not Imposter/Double Agent)
  • Game-specific instructions

No marketing or promotional content is included.

Technical Implementation

The opt-in process is implemented in the mobile app as follows:

// User selects SMS delivery method if (deliveryMethod === 'sms') { // Display consent language showConsentMessage(); // Show phone number input fields showPhoneInputs(); // User must manually enter phone numbers // No automatic collection or pre-filling // Only send after explicit "Send Cards" action if (userTapsSendCards()) { sendSMSMessages(); } }

Compliance Statement

Khafī (Muslim Imposter) complies with TCPA (Telephone Consumer Protection Act) requirements:

  • Users provide express written consent before receiving SMS messages
  • Consent is obtained through clear, unambiguous language
  • Users are informed of message and data rates
  • Opt-out instructions are provided and honored
  • Messages are sent only for the specific purpose users consented to
